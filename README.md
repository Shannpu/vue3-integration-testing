# vue3-integration-testing

Vue3 框架下的集成测试，使用 Vitest 和 Vue Test Utils。

## Run

### Run project

```sh
npm install
npm run dev
```

### Build project

```sh
npm run build
npm run preview
```

### Run Testing [Vitest](https://vitest.dev/)

```sh
npm run test
npm run coverage
```
